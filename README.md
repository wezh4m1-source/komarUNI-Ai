Komar Uni AI is an intelligent chatbot built for students and faculty at **KUST (Komar University of Science and Technology)**. Powered by GPT-2 and advanced AI technology, it provides research-backed answers to academic questions.

## ✨ Features

- 🤖 **AI-Powered Responses** - Using GPT-2 transformer model
- 📚 **Research Capability** - Access to comprehensive knowledge base
- 💬 **Real-time Chat** - Instant responses 24/7
- 📖 **Knowledge Base** - Python, AI, Machine Learning, KUST topics
- 🔍 **Source Citations** - See where answers come from
- 💾 **Chat History** - Track all conversations
- 🎓 **KUST Integration** - University-specific information
- 📱 **Responsive Design** - Works on desktop and mobile
- ⚡ **Fast & Reliable** - Minimal dependencies
